AJAX Reviews

Version 1.2.4

The AJAX Reviews Magento extension allows customers to leave product reviews without reloading the page or redirecting to a separate feedback form. After leaving a comment, a customer can quickly continue shopping from that place where s/he stopped to write a comment. Flexibility of the backend options will help you to tailor the reviews block style to your store design. The content of the dialog windows and the color of rating stars are also customizable.

The extension also allows sending notifications of pending reviews to admins and mails after purchase with a feedback form to customers.

Features

  • Ability to insert reviews into a custom place
  • Ability to add rich snippets markup
  • Admin notification of a pending review
  • Responsive post-purchase email template
  • Ability to preview a post-purchase email from Sending Queue
  • Ability to add a review from post-purchase email in two clicks
  • Customer mail after purchase
  • Mails after purchase for existing orders
  • Ability to preview and send a test version of mail after purchase
  • Ability to send customer notification in any order status
  • Ability to cancel a customer notification
  • Automatic post-purchase mails cancel after order cancel
  • Automatic post-purchase mail cancel after item return
  • Automatic mail cancel if a customer already left a review
  • Ability to cancel mails for one item from different order
  • Auto save of a non-submitted review if the popup was closed
  • Ability to display rating as stars or hearts
  • Color icons for non-Latin nickname letters
  • Editable button label for a first review
  • A link to leave a first review next to a product name
  • Customizable “How to write an awesome review?” dialog
  • Customizable “Thank You” dialog
  • Gravatar images
  • Ability to sort reviews
  • Customizable reviews style
  • Ability to share reviews
  • Ability to vote for reviews
  • Changeable color of rating figures
  • Intuitive interface
  • Easy 2-minutes installation
  • Meets Magento programming practices for most versions and customizations compatibility

Installation

Before you start, we would like to point out that if you are not confident performing the installation yourself, you should let the team of Magpleasure Ltd. do it. People who developed the extension would be the best choice and they usually offer this service for a small fee. See http://www.magpleasure.com/installation-service.html.

Backup your data

Make a Backup copy of the files and database for your Magento store.

Disable Compilation

If you have enabled compilation, please disable it. Extension installation with enabled compilation will inevitably lead to problems in the frontend or the backend. This can be corrected, but would require an expert help. To avoid it, temporary disable the compilation.

Enable all Types of Cache

If you have disabled the cache, then turn it on and refresh it. It is best to install extensions to the store when no one except you has access to it during that time, because installation process can begin before all extensions files are loaded. This may lead to some problems.

Download and unzip the package

Download the extension package from My Downloadable Products in the customer area and unzip it to the store directory.

Additional installation steps

Some extensions require from you making additional modifications to the source files of your store. Find the Configuration below this one. There you will find additional configuration steps.

Refresh Store Cache

Unless you refresh the store cache, the extension will not be installed properly. Go to the System → Cache Management backend page and click Flush Cache Storage. The cache is cleared and the extension will be installed to the store after refreshing any page.

That's done

The extension is installed to your store and you can start configuring it if you need. If you were using compilation, you can enable it now.

Configuration

Extension Configuration

To configure the extension, go to the Catalog → Reviews and Ratings → AJAX Reviews → Configuration or System → Configuration → AJAX Reviews backend page. Here you can configure display settings, style settings, dialogs options, admin notifications of pending reviews and mails after purchase options (see the Configure AJAX Reviews section in the User Guide).

Mails after Purchase Management

To manage customer mails after purchase, navigate to the Catalog → Reviews and Ratings → AJAX Reviews → After Purchase Emails backend page. Here you can send post-purchase emails or or cancel them if necessary (see the After Purchase Emails Management section in the User Guide).

Reviews Integration into a Custom Place

If you want to integrate the reviews block into a custom place, you can find the detailed instructions in the How to Insert Reviews into a Custom Place section of the User Guide.

Troubleshooting

After the extension installation the store gives an error, or blank page, or suggests to start Magento installation procedure.
Change the owner of the extracted extension files to the web server user and set 775 permissions on them. Clear the store cache and try again.
There is no MagPleasure extensions under my configuration section, or having the extension tab clicked I get a blank page, or Access Denied error.
Clear the store cache, browser cookies, logout and login again.
I've set up everything correctly, inserted the HTML code but there is nothing on that page.
Clear the store cache, clear your browser cache and domain cookies and refresh the page.
My configuration changes do not appear on the store.
Clear the store cache, clear your browser cache and domain cookies and refresh the page.
